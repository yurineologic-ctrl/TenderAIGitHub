## Project Overview

**TenderAIGitHub** — веб-парсер для сбора данных о ноутбуках с сайта Lenovo Ukraine (https://shop.lenovo.ua).

Парсер:
- Автоматически загружает страницы каталога ноутбуков
- Извлекает детальные характеристики из страниц товаров (CPU, RAM, GPU, дисплей, порты, сеть и т.д.)
- Сохраняет собранные данные в Excel файл на рабочем столе (`NOUT.xlsx`)
- Поддерживает украинский язык в спецификациях

## Setup

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

**Основные зависимости:**
- `playwright` — веб-браузер для JavaScript-heavy сайтов
- `beautifulsoup4` — парсинг HTML
- `openpyxl` — создание Excel файлов
- `pandas` — работа с табличными данными

### 2. Установка браузера для Playwright (первый запуск)

```bash
playwright install chromium
```

## Running the Parser

### Запуск основного парсера

```bash
python lenovo_parser.py
```

**Параметры в коде (верх файла):**
- `BASE_URL` — базовый адрес сайта
- `CATALOG_URL` — ссылка на каталог ноутбуков
- `MAX_ITEMS` — максимальное количество товаров для сбора (по умолчанию 100)
- `PAGE_TIMEOUT` — таймаут загрузки страницы (30000 мс = 30 сек)
- `LOAD_DETAIL_PAGES` — загружать ли детали со страниц товаров (по умолчанию `True`)
- `OUTPUT_PATH` — путь для сохранения Excel файла

### Результат

Файл `NOUT.xlsx` на рабочем столе с колонками:
- Серия, модель, Part Number
- Характеристики процессора (бренд, модель, ядра, потоки, частота, кеш)
- ОЗУ (тип, частота, размер, количество слотов)
- Накопитель (SSD/HDD)
- Дисплей (диагональ, разрешение, тип матрицы, яркость, контраст, время отклика, частота обновления)
- Видеокарта
- ОС
- Звук, камера, порты (USB, HDMI, DisplayPort)
- Клавиатура (влагозащита, украинская раскладка, тачпад)
- Сеть (3G/4G, Bluetooth, WiFi, LAN)
- Гарантия, цена, ссылка на товар

## Key Implementation Details

### Извлечение спецификаций

Основная логика парсинга в `lenovo_parser.py` использует:

1. **Регулярные выражения** — скомпилированные паттерны (RE_* константы) для извлечения значений из текста спецификаций
2. **BeautifulSoup** — парсинг HTML структуры страниц
3. **Playwright** — навигация по сайту, обработка JavaScript-контента, клики на "Далее" для пагинации

### Обработка данных

- **Украинский язык**: спецификации содержат украинский текст, регексы учитывают особенности языка (например, "дюйм" vs "дюйма")
- **Форматирование Excel**: стилизация ячеек, выравнивание, шрифты, цветовое выделение (в коде используется openpyxl)
- **Маппинги**: `SERIES_MAP` (ThinkPad, Legion и т.д.), `RESOLUTION_MAP` (FHD → 1920x1080)

### Оптимизация

См. `OPTIMIZATION_GUIDE.md` для рекомендаций по улучшению скорости и качества парсинга.

## Common Tasks

### Добавить новую спецификацию для сбора
1. Добавьте регулярное выражение в раздел `RE_*` констант
2. Добавьте новую колонку в `COLUMNS`
3. Добавьте логику извлечения в функцию парсинга (обычно `extract_*` функции)

### Изменить количество товаров
Измените `MAX_ITEMS` в верхней части `lenovo_parser.py` на нужное значение.

### Сохранить результаты в другое место
Измените `OUTPUT_PATH` в верхней части `lenovo_parser.py`.

### Запустить на другом сайте Lenovo
Измените `BASE_URL` и `CATALOG_URL` для другой страны/сайта.

## Performance Notes

- Парсер работает медленно по дизайну — сайт требует времени на загрузку (PAGE_TIMEOUT = 30 сек)
- Для больших объёмов данных (MAX_ITEMS > 500) может потребоваться несколько минут
- Количество запросов контролируется `MAX_ITEMS` для предотвращения блокировки IP
- При ошибках сети парсер отправляет исключение (нет автоматических повторов)
